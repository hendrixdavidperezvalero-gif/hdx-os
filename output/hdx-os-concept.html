Lo primero de la mañana

El ritual es el producto.

Un tablero que solo se mira, se abandona. Este se escribe. Dos rituales — del command center de Notion y del método Make Time — sostienen todo lo demás.

Ritual uno

El botón «Nuevo día»

Lo presionas y genera el checklist del día: la rutina fija + el paso actual de cada plan vivo. No decides qué hacer, solo haces.

Es el gancho que reemplaza al susto de la empresa: abres el OS porque presionarlo es cómo empieza tu día.

Ritual dos

Los tres campos + un número

Se escriben, no se leen. Te obligan a estar en el OS cada mañana.

El corazón

Un plan es una página. Un paso es una casilla.

Escribes el objetivo en Obsidian con calma. El OS calcula el paso actual — el primer - [ ] sin marcar — y lo pone en tu Hoy. Nada más.

Qué construimos primero (v1)

Lo que mata a un sistema así no es que le falten funciones — es que el v1 sea tan grande que nunca lo termines. El MVP es deliberadamente pequeño: una pantalla, un fin de semana.

El motor, sin misterio

El vault es la única base de datos.

El Sheet no es un segundo lugar donde vive algo — es un buzón, un tubo entre tu teléfono y tu PC. La app nunca escribe tu markdown; de eso se encarga Claude Code.

La verdad

El vault

Planes, áreas, diario, notas. Markdown en GitHub. La app lo lee.

La cara

La app

Lee el vault. Escribe solo al buzón. Nunca toca tu markdown.

El que barre

Claude Code

Vacía el buzón al vault, rutea capturas, resume videos. Único escritor.

dia

Cómo empezamos

Stack, prerrequisito y primeros pasos.

Stack

Next.jsVercelGoogle Sheets PWALogin PINSin API de IA

Reutiliza la capa de datos de belen-web (VTR OS) — leer Google Sheets ya está resuelto. La IA la hace Claude Code por lotes, así que el MVP no gasta un centavo de API.

El único precio: el vault a GitHub

La app vive en Vercel; el vault en tu PC. GitHub es el único canal por el que pueden verse. Sin eso, combinar la app con el vault es imposible por definición.

🔓 La app lee tus planes, áreas y log 🔓 Claude Code desde el móvil, con la PC apagada
